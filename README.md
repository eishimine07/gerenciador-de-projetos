## Descrição

Ferramenta onde o usuário final pode visualizar, adicionar, editar, remover e favoritar projetos

## Tech Stack
- Vue 3
- Vite
- Pinia
- Typescript
- Vue I18n
- Vuetify
- Vitest

## Rodando o projeto
Instalando as dependências
```bash
npm install
```

Iniciando o server
```bash
npm run dev
```
